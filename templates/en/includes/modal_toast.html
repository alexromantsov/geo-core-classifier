Instructions for Using the "Core Description" Module

This module allows you to analyze the textual description of a core to extract key information. However, it is important to consider the following limitations:

  • Word Count Limit: For users without the PRO version of the module, there is a limit of 25 words per request.
  • Language Limitations: The text processing algorithm is optimized for working with texts in English.

Ensure that your description meets these requirements for optimal processing and data analysis.
