Инструкции по использованию модуля "Описание керна"

Данный модуль позволяет анализировать текстовое описание керна для извлечения ключевой информации. Однако, важно учитывать следующие ограничения:

  • Ограничение на количество слов: Для пользователей без PRO версии модуля действует ограничение в 25 слов за один запрос.
  • Языковые ограничения: Алгоритм обработки текста оптимизирован для работы с текстами на русском языке.

Убедитесь, что ваше описание соответствует этим требованиям для оптимальной обработки и анализа данных.
